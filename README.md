# Site-Images

Organized image hosting repository for [The Back Room](https://github.com/The-Back-Room) forums.

## Directory Structure

| Folder | Purpose |
|---|---|
| `avatars/` | User profile avatars |
| `banners/` | Forum and section banners |
| `backgrounds/` | Page and section background images |
| `icons/` | Site icons and UI elements |
| `emotes/` | Forum emoji and reaction images |
| `misc/` | Other images that don't fit the categories above |

## Supported Formats

The following image formats are accepted:

- **PNG** (`.png`) — preferred for icons, emotes, and images with transparency
- **JPEG** (`.jpg` / `.jpeg`) — preferred for photographs and large backgrounds
- **GIF** (`.gif`) — accepted for animated emotes and avatars
- **WebP** (`.webp`) — accepted for all categories; recommended for optimized file sizes
- **SVG** (`.svg`) — accepted for icons only

## Usage

Images in this repository are referenced via their raw GitHub URL:

```
https://raw.githubusercontent.com/The-Back-Room/Site-Images/main/<folder>/<filename>
```

**Example:**

```
https://raw.githubusercontent.com/The-Back-Room/Site-Images/main/emotes/wave.png
```
